# Janglim マニュアルへようこそ

Janglim は **.NET 用の LR/LALR パーサージェネレータエンジン** です。\
言葉が難しいですよね？\
**大丈夫です。**\
このマニュアルは
あの難しい単語を一つも知らないという前提で、本当に最初からゆっくり解きほぐしていきます。

## このマニュアルは誰のためのもの？

**パーサーとは何か、コンパイラがどう動くのか、一つも知らなくても大丈夫です。**\
このマニュアルは「なぜこれが
必要なのか」から一歩ずつ、できるだけやさしい言葉で説明します。\
だから読み終える頃には、二つのものが手に残ります。

1. **構文解析という概念** そのもの — どんなパーサーを使うにしても一生役立つ知識
2. **Janglim という道具** — そして（望めば）その中身がコードでどのように作られているのかまで

## このマニュアルは二筋に分かれています

負担を減らすために **基本 / 発展** の二つのコースに分けました。

**🟢 基本コース — 概念だけを気楽に。**\
公式もコードもなしに、*概念と直感* だけをつかみます。\
各ページは:

| 段階 | 何を |
|---|---|
| **① なぜ必要なのか** | これがないと何ができないのか（問題から） |
| **② 何をするのか** | 概念そのもの + 小さな例 |
| **③ プレイグラウンドで見る** | ブラウザで直接動かしながら確認 |

**🔵 発展コース — 公式 + コード。**\
同じ概念を *どんな文法にも通用する公式（アルゴリズム）* として整理し、
それが **Janglim のコードでどう実装されているのか** まで見ます。\
基本コースの各概念の最後に「→ 発展へ」
というリンクを付けておきます。

> **初めてなら基本コースだけを通して読んでください。**\
> 発展は後で気になったときに見ても十分です。🙂

## 一つの例で最後まで進みます

マニュアルはこの **小さな文法一つ** を主にずっと使います — 足し算・掛け算のある小さな算術式です。

<pre class="lrbox">
<span class="nt">Expr</span>   : <span class="nt">Expr</span> <span class="setm">'+'</span> <span class="nt">Term</span> | <span class="nt">Term</span> ;
<span class="nt">Term</span>   : <span class="nt">Term</span> <span class="setm">'*'</span> <span class="nt">Factor</span> | <span class="nt">Factor</span> ;
<span class="nt">Factor</span> : <span class="setm">'('</span> <span class="nt">Expr</span> <span class="setm">')'</span> | <span class="setm">id</span> ;
<span class="setm">id</span>     := "[a-zA-Z]+" ;
</pre>

そしてこの入力をずっと構文解析していきます: **`a + a * a`**

各チャプターがこの *同じ* 例に理解を一枚ずつ重ねていきます。\
ときどき、この文法では見せられない概念（ε、衝突など）が出てきたら、その場で *小さな補助文法* をまるごと別に見せます。\
（この文法の読み方は初めて出てくるとき
一緒にゆっくり解きほぐしていくので、前もって心配しなくて大丈夫です。）
